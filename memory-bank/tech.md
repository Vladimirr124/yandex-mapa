Текущий стек проекта
=====================

- HTML + CSS (чистая вёрстка без сборки)
- JavaScript без фреймворков (vanilla JS, ES5-стиль, один `index.html`)
- Яндекс.Карты JavaScript API 2.1 (подключение по CDN в `<script src="https://api-maps.yandex.ru/2.1/...">`)
- Статический локальный сервер для разработки: `npx serve .` из папки `demo-point-layer`

Важно: стек считать зафиксированным; менять его можно только с явного согласия пользователя.
