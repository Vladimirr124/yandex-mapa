Координаты

Наведите курсор на карту

0° — горизонтально. Для путей по диагонали введите 30–50° (например 35).

Рельсы (пути в депо)

Клик по карте подставляет координаты: первый клик — начало, второй — конец. Или введите вручную (широта, потом долгота; дробная часть через точку или запятую).

Последний скриншот (распознавание)

Не загрузилось. Откройте страницу по http:// (не file://) и запустите сервер скриншотов.

Сервер: python gpu+out/serve_last_screenshot.py — порт 8766

При распознавании (camall_screen_capture.py) поезд появляется на карте при входе и удаляется при выходе из депо (WebSocket порт 8765).
