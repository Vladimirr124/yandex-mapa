# Пробный скрипт: слой точек на Яндекс.Карте

**Цель:** понять, как работают слои в Яндекс.Картах и как ими пользоваться (локальный доступ, ограниченное управление).

## Как запустить

1. **Ключ API.** В `index.html` в строке с подключением скрипта замени `YOUR_API_KEY` на свой ключ Яндекс.Карт:
   - Получить ключ: [Яндекс.Разработчики](https://developer.tech.yandex.ru/) → JavaScript API и HTTP Геокодер.
2. Открой `index.html` в браузере (лучше через локальный сервер, например `npx serve .` в папке `demo-point-layer`).

## Как это устроено

- **Слой = одна коллекция.** Все «наши» точки лежат в `GeoObjectCollection` с именем `pointsLayer`. Карта получает только эту коллекцию (`map.geoObjects.add(pointsLayer)`), остальные объекты карты мы не трогаем.
- **Доступ локальный.** Работа с точками только через `pointsLayer`: `pointsLayer.add(placemark)` и `pointsLayer.remove(placemark)`. Других ссылок на точки нет — управление ограничено этим слоем.
- **Кнопка «Добавить точку»:** создаётся метка с фиксированными координатами, добавляется в `pointsLayer`, через 10 секунд та же метка удаляется из `pointsLayer`.
- Координаты точки заданы в коде: `POINT_COORDS = [55.7558, 37.6173]` (Москва). Можешь поменять под свои нужды.

## Что проверить

- Нажал «Добавить точку» → точка появилась.
- Через 10 секунд точка исчезла.
- Можно несколько раз нажать подряд — каждая точка живёт свои 10 секунд и удаляется своей таймер-функцией.
